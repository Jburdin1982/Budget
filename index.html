⚙️ One-Time Firebase Setup — Enables real-time sync between your phone and Elizabeth's

You need a free Firebase Realtime Database. It takes about 5 minutes:

  1. Go to console.firebase.google.com → sign in with Gmail → click Add project → name it burdin-budget → click through to finish
  2. In the left menu click Build → Realtime Database → Create database → pick any location → choose Start in test mode → Enable
  3. Copy the database URL shown (looks like https://burdin-budget-xxxx-default-rtdb.firebaseio.com)
  4. Click the ⚙️ gear (Project settings) → scroll to Your apps → click </> → Register app → copy the apiKey value
  5. Paste both values below and click Connect. Then share this same page URL with Elizabeth — she enters the same two values on her device and you're synced.
